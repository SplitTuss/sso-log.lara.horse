# sso-log.lara.horse

This is a [Next.js](https://nextjs.org) project with [IndexedDB](https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API)

## Quickstart

First, install dependencies:
```bash
yarn
```

Then, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment
Push code to GitHub pages:

```bash
yarn deploy
```
